Digital
pinOn turn a pin HIGH
pinOff turn a pin LOW
readPin read digital pin value
Analog
analogWrite write analog value 0-255
analogRead read analog pin A0-A5
Time
wait pause for milliseconds
waitMicro pause microseconds
Serial
serialStart begin serial monitor
serialPrint print text to monitor
serialPrintLine print + new line
