Smart Plant Watering System

Build an automatic plant watering system that detects soil moisture and waters the plant only when needed.

Build time3–4 hours
DifficultyIntermediate
PlatformArduino

Overview

The system continuously checks the soil moisture level using a moisture sensor. When the soil becomes dry, the Arduino automatically turns on a solenoid valve to water the plant. Once enough moisture is detected, the solenoid valve close automatically.

Step-by-Step Build Guide

1

Prepare the plant setup

Place the soil moisture sensor into the plant pot soil. Ensure the sensing area is fully covered by soil but avoid pushing too deep.

2

Connect the soil moisture sensor

Connect the soil moisture sensor to the Arduino as shown in the wiring diagram. The sensor should be placed in the plant pot soil, ensuring the sensing area is fully covered but not submerged.

3

Connect the relay module

Connect the relay input pin to Arduino D7. The relay acts as a switch to safely control the solenoid valve.

4

Wire the solenoid valve

Connect the solenoid valve to the relay output. Ensure the valve is properly connected to the water source and the plant's irrigation system.

5

Upload the Arduino code

Open Arduino IDE and upload the following code:


  int sensorPin = A0;
  int relayPin = 7;

  int moistureValue = 0;
  int dryThreshold = 600;

  void setup() {
    pinMode(relayPin, OUTPUT);
    digitalWrite(relayPin, HIGH);

    Serial.begin(9600);
  }

  void loop() {

    moistureValue = analogRead(sensorPin);

    Serial.println(moistureValue);

    if (moistureValue > dryThreshold) {
      digitalWrite(relayPin, LOW); // Pump ON
    }
    else {
      digitalWrite(relayPin, HIGH); // Pump OFF
    }

    delay(1000);
  }
6

Test & troubleshoot

Power the Arduino and monitor the soil condition. Adjust the dryThreshold value depending on your soil type.
