The Learning Path

Three Stages to Innovation

A clear progression from beginner curiosity to building connected smart systems.

1
Stage 1 · Beginner 🔦

The Explorer

Building real circuits from day one. Students learn to ask "how does this work?" and find the answer by making it.

  • LED circuits & resistors
  • Light & distance sensors
  • Breadboard prototyping
  • Block-based visual coding
  • Introduction to Arduino
  • Basic science experiments
Ages 10–12 · No experience needed
2
Stage 2 · Intermediate 🤖

The Maker

From circuits to machines. Students build robots that move, sense, and respond — writing real code to control them.

  • DC motors & servo motors
  • Line-following robots
  • Obstacle-avoidance systems
  • C++ logic programming
  • Remote-controlled vehicles
  • Engineering design cycle
Ages 12–15 · Stage 1 or equivalent
3
Stage 3 · Advanced 🌐

The Innovator

Internet of Things and smart systems. Students build devices controlled from a smartphone, anywhere in the world.

  • ESP32 & Wi-Fi connectivity
  • IoT dashboards & mobile control
  • Python & web fundamentals
  • Cloud data & automation
  • Smart agriculture projects
  • Showcase capstone project
Ages 15–17 · Stage 2 or equivalent
