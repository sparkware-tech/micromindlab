Sensor Color Detection

How TCS230 TCS3200 Color Sensors Work

The TCS230 TCS3200 Color Recognition Sensor Module detect the color of an object by measuring the intensity of red, green, and blue (RGB) light reflected from its surface.

Detects RGB Colors
Output Frequency
Uses Color Recognition

Simple Idea

The TCS230 TCS3200 is color recognition sensor that identify colors by measuring reflected light. The sensor contains arrays of red, green, blue, and clear photodiodes.

When light reflects from an object, the sensor measures how much red, green, and blue light is received. A microcontroller then compares these values to determine the object's color.

How It Works

1

Illuminate the Object

White LEDs on the module shine light onto the object's surface.

2

Light is Reflected

Different colors reflect different amounts of red, green, and blue light.

3

Photodiodes Measure Color

The sensor selects red, green, or blue filters and measures the intensity of the reflected light.

4

Frequency Output is Generated

The measured light intensity is converted into a square-wave frequency signal.

5

Color is Identified

The microcontroller compares red, green, and blue values and determines the object's color.

Key Features

  • Detects red, green, blue, and clear light components.
  • Frequency output is easy to read using microcontrollers.
  • Built-in white LEDs improve sensing accuracy.
  • Compatible with Arduino, ESP32, Raspberry Pi, and STM32.
  • Fast and reliable color recognition.

Main Pins

VCC – Power supply (5V)

GND – Ground connection

OUT – Frequency output signal

S0, S1 – Output frequency scaling selection

S2, S3 – Color filter selection

OE – Output enable pin

The Arduino reads the frequency output from the sensor for the red, green, and blue filters. These values are then used to determine the detected color.

Example Arduino Code


    #define S0 4
    #define S1 5
    #define S2 6
    #define S3 7
    #define sensorOut 8

    int redFrequency = 0;
    int greenFrequency = 0;
    int blueFrequency = 0;

    void setup() {
        pinMode(S0, OUTPUT);
        pinMode(S1, OUTPUT);
        pinMode(S2, OUTPUT);
        pinMode(S3, OUTPUT);
        pinMode(sensorOut, INPUT);

        digitalWrite(S0, HIGH);
        digitalWrite(S1, LOW);

        Serial.begin(9600);
    }

    void loop() {
        // Read RED
        digitalWrite(S2, LOW);
        digitalWrite(S3, LOW);
        redFrequency = pulseIn(sensorOut, LOW);

        // Read GREEN
        digitalWrite(S2, HIGH);
        digitalWrite(S3, HIGH);
        greenFrequency = pulseIn(sensorOut, LOW);

        // Read BLUE
        digitalWrite(S2, LOW);
        digitalWrite(S3, HIGH);
        blueFrequency = pulseIn(sensorOut, LOW);

        Serial.print("R = ");
        Serial.print(redFrequency);

        Serial.print(" G = ");
        Serial.print(greenFrequency);

        Serial.print(" B = ");
        Serial.println(blueFrequency);

        delay(500);
    }

Open the Serial Monitor to view the Red, Green, and Blue values. By comparing these values, the Arduino can identify different colors such as red, green, blue, yellow, black, and white.

Common Uses

Color sorting machines, smart recycling systems, robotic vision, industrial automation, line-following robots, object classification, quality control systems, and educational electronics projects.
